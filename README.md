# Тестовое задание

### Инструкция для запуска

Запуск приложения локально:

1. Запуск контейнеров
```bash
docker-compose up -d
```

2. Выполнним миграции и накатим сиды
```bash
docker-compose exec app php artisan migrate:fresh --seed
```

3. Запуск тестов локально (запускается в тестовой базе)
```bash
docker-compose exec app php artisan test
```

Проект будет доступен по http://localhost/ или http://127.0.0.1:80
